# Sistema de Inventario en Python

Este es un programa de consola para gestionar productos en una base de datos SQLite.

## Funcionalidades

- Agregar productos
- Mostrar todos los productos
- Buscar por nombre o ID
- Actualizar productos
- Eliminar productos
- Validaciones de ingreso con funciones personalizadas
